### Week 3 Achievements:

* Understood the concept of cloud computing and the overall structure of AWS services.

* Identified and differentiated major AWS service groups such as Compute, Storage, Networking, and Database.

* Successfully created and configured an AWS Free Tier account.

* Became familiar with AWS Management Console, including navigating services and managing basic resources.

* Installed and configured AWS CLI, including:

  * Access Key
  * Secret Access Key
  * Default Region
  * Output format

* Used AWS CLI to perform basic operations such as:

  * Checking account and configuration information
  * Listing available AWS regions
  * Viewing EC2 resources
  * Creating and managing key pairs
  * Monitoring running services

* Gained hands-on experience with Amazon EC2:

  * Launching instances
  * Connecting via SSH
  * Attaching and managing EBS volumes

* Understood AWS Networking fundamentals:

  * VPC structure and subnet design
  * Differences between Security Groups and Network ACLs
  * Roles of Internet Gateway and NAT Gateway

* Successfully practiced building a basic multi-tier network architecture.

* Developed the ability to manage AWS resources using both the AWS Management Console and AWS CLI in parallel.
